## .{w|h}-{size}

If 1 rem = 16px (default font size of the document), with taillwind we need to divide by 4

4 in tailwind = 1 rem


| Size             |                     |
|------------------|---------------------|
| 0,1,2,3,4,4,5,6  | +1                  |
| 8,10,12,         | +2                  |
| 16,20,24,        | +4                  |
| 32,40,48,64      | +8                  |
| 1/2 ...          |                     |
| 1/ {3,4,5,6,12}  |                     |
| screen, full     |                     |
